# Contribution Guidelines

This repository curates **research skill development opportunities in academic settings**, including mentorship programs, research-oriented internships, summer schools, fellowships, and predoctoral training initiatives.

To add, remove, or update an entry, please submit a pull request that follows the guidelines below:

- Include only **active and verifiable programs** focused on developing research skills for predoctoral students (undergraduate, post-baccalaureate, master's, or predoctoral levels).
- Ensure the program is **academically affiliated** (e.g., universities, research institutes, academic research labs, or faculty-led initiatives).
- Link to the **official program webpage**.
- Verify that links are accessible and avoid duplicate entries.
- Organize entries by **continent**, and maintain **alphabetical ordering within each region**.
- Use the program’s official name as the hyperlink text.
- Provide a **concise, neutral, and non-promotional description**.
- Exclude industry-only programs, generic job postings, and non-research opportunities.
- When available, include funding, duration, and eligibility information.

## Updating Your Pull Request

If revisions are requested, please update the existing pull request instead of opening a new one. Keeping discussion and revisions within the same PR helps streamline review and merging.
