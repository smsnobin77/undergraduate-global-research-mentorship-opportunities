# Contribution Guidelines

This repository curates **research skill development opportunities in academic settings**, including mentorship programs, research internships (remote or in person), summer schools, and predoctoral training initiatives.

To add, remove, or update an entry, submit a pull request that adheres to the following:

- Include only **active and verifiable programs** focused on developing research skills for predoctoral students (undergraduate, post-baccalaureate, or early graduate levels).
- Ensure the program is **academically affiliated** (e.g., universities, research institutes, or faculty-led initiatives).
- Link to the **official program webpage**.
- Verify that links are accessible and avoid duplicates.
- Maintain **alphabetical ordering by program name**.
- Use the programâ€™s official name as the hyperlink.
- Provide a **concise, neutral, non-promotional description**.
- Exclude industry-only programs, generic job postings, and non-research internships.

## Updating Your Pull Request

If revisions are requested, edit the existing pull request rather than opening a new one. Update the PR directly so it can be reviewed and merged efficiently.
